Comandos Básicos do Git

  • git init -- inicia o monitoramneto da pasta que será o repositorio local
  • git add "nome de arquivo" -- adiciona o arquivo especifico na fase stage
  • git add . -- adiciona todos os arquivos na fase
  • git commit -m "frase para identificar a ação -- commit envia para a fase "
